## インタビュー時の傾聴スタンス（リサーチ概論）

出典：杉原保史(2012)『技芸としてのカウンセリング』創元社に田中が加筆。

1. **ありのまま、ただ聞く**
   - 仮説禁欲：勝手に仮説をつくらない
   - 対象者になりかわり、対象者の主観を追体験しようとする
   - 勝手に判断しない
   - 気づいても、そのままにしておく

2. **がんばらないで聴く**
   - 気負わない
   - 沈黙も大切にする
   - 相手を受け入れる、答えを聴こうと無理しない

3. **体験を聴く**
   - 対象者に体験を内的に探求させる

4. **無知の姿勢で聴く**
   - 知っていることでも好奇心を持って聞く

5. **声を聴く／態度や様子／姿勢を聴く**
   - 声の状況と内容はしばしば「ズレる」
   - 対象者の思考や感情は体に現れる

6. **問題を味わう**
   - すぐに診断したり、結果を出したりしないで味わう
   - 答えを誘導しないこと

## インタビューの本質

> インタビューとは？
> **異なる人間同士の間で、同じものを、見ることができている状態**

- なるべく具体的に、相手が思い描いている世界を聞き出し、
- 自分の脳裏にも、同じような光景を描き、お互いに見える世界を共有する

> **相手が経験してきた世界、相手が感じてきた感情のありようを追体験し、さぐっていく行為。**

## ペイン／インサイト（氷山モデル）

**ペイン（水面上）**
- 表層的な欲求／ニーズ
- ユーザー本人も自覚している課題

**インサイト（水面下）**
- 真の欲求
- 劣等感／コンプレックスに隠れていることも多い
- 欲求＋解決策の方向性がセット
- ユーザー本人は自覚できていない

## Whyを言及する、曖昧さを見逃さない（インタビュー設計）

**POINT**：「分かりやすい」や「心地よい」のように人によって捉え方が異なる回答は、具体的に深掘る質問を続けましょう。

例：
- 回答者「○○が分かりやすいからです。」
- 質問者「分かりやすいと思ったポイントを詳しく教えてください。」

## 出来る限り「なぜ」と直接問いかけない（インタビュー設計）

| 5W1H | なぜ→言い換え |
|------|-------------|
| いつ | なぜそうしているのですか？ → **いつからそうしているのですか？** |
| 何を | なぜそれをする決め手は何でしたか？ → **それをする決め手は何でしたか？** |
| 何を | なぜそれをしないのですか？ → **何が妨げになっているのですか？** |
| どこで | なぜそう感じたのですか？ → **どこでそう感じたのですか？** |
| どのように | なぜそう思うのですか？ → **どのようにしてその考えに至ったのですか？** |

## インタビュー時の深ぼるポイント：例）ワクワク

- 一次情報＝個人的実体験：どんなときにワクワクを感じるか？
- 状況：ワクワクしたときの状況は？
- 理由：そのとき、なぜワクワクしたのか？
- 時・条件：ワクワクする「時・条件」は？
- 類義語／違い／共通点：ワクワクと似ている言葉は？　違いや共通点は？
- 反対語：ワクワクの反対語は？
- 構成要素：ワクワクの構成要素は？
- 変化や進化：10年前といまと比べてワクワクの意味は異なる要素があるのか？
- 定義：ワクワクの定義は？

## 人によって捉える現実や解釈が違う（リサーチ概論・意味の解釈）

「コップの中に水が半分入っている」という具体的な事実に対して：
- 「まだ半分ある」と考える人
- 「もう半分しかない」と考える人
- 「お酒が飲みたいのに、水しかない」と考える人

それぞれがその人にとっての**現実**であり、これが**意味の違い**である。
