# 要件定義：異世界ステータス診断アプリ
作成日：2026-04-11

## 概要
6問の選択式質問に答えると、異世界転生したときのステータス画面が生成されるアプリ。

## ユーザー・用途
- 対象：自分（たくやさん）のみ（プライベートツール）
- 将来：クライアント・受講生への展開も視野

## スタック
- TypeScript / React / Vite
- Tailwind CSS v3
- ローカル実行（`npm run dev`）のみ

## 画面構成
| 画面 | 内容 |
|---|---|
| スタート画面 | タイトル・「診断開始」ボタン |
| 質問画面 | 6問をA/B/C/Dで1問ずつ表示 |
| ローディング画面 | 「異世界転生中...」演出（プログレスバー） |
| 結果画面 | なろう系ステータスウィンドウ風に全ステータス表示 |

## デザイン方針
- なろう系・アニメ風ステータスウィンドウ
- 青・金・紫を基調
- ステータスバーにアニメーション（数値が伸びていく）
- 星フィールド背景
- もう一度やるボタンでリセット

## ステータス項目（21項目）
### 生命系：HP / MP / SP
### フィジカル系：STR / AGI / VIT / DEF
### 知性系：INT / WIS / MND
### 対人系：CHA / EMP / NGT / LDR
### 技術・創造系：TEC / CRE / ANA / STZ
### その他：LUK / FOC / ADP

## 質問→ステータス対応
| 問 | 質問 | 変動ステータス |
|---|---|---|
| Q1 | 体力 | STR / VIT / MP |
| Q2 | 運 | LUK / ANA |
| Q3 | 並走 | FOC / SP / [複数管理] |
| Q4 | 問いを立てる | [問いを立てる] / WIS |
| Q5 | 最大の強み | カテゴリ最高値・称号・★ |
| Q6 | 批判への対処 | DEF / MND / INT |

## ファイル構成
```
src/
  App.tsx                  # 画面ルーティング
  index.css                # Tailwind + Google Fonts
  logic/
    types.ts               # 型定義
    questions.ts           # 質問データ
    calculator.ts          # ステータス計算ロジック
  components/
    StarField.tsx          # 背景の星エフェクト
    StartScreen.tsx        # スタート画面
    QuestionScreen.tsx     # 質問画面
    LoadingScreen.tsx      # ローディング画面
    ResultScreen.tsx       # 結果画面
```
