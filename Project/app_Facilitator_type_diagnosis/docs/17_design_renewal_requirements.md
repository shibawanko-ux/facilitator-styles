---

## デザインコンセプト

**「イラストが主役のクリーン＆フレンドリーなデザイン」**

- 白ベース＋ブルーアクセントのライトなデザイン（ダーク背景はNG）
- フラットイラスト（キャラクター）がブランド感と温かさを生む
- 余白たっぷり・テキストは少なめ
- ブルーはセクション背景として部分的に使う

---

## カラー方針

| 用途 | 方針 |
|------|------|
| 全体背景 | 白（#ffffff / #fafafa） |
| セクション区切り | ブルー系（現行 primary カラーを維持） |
| テキスト | 深いネイビー系（slate より落ち着いた印象） |
| アクセント | ブルー（現行 primary カラーを維持） |

## イラスト方針（最重要）

- **フラットイラスト × マンガ風キャラクター**
- 16タイプそれぞれに1体ずつキャラクターを用意
- スタイル：ichizenbooks.jp の人物イラストと同系統（フラット・親しみやすい・表情あり）
- 用途：
  - ヒーローセクションに大きく配置
  - 16タイプカードに各キャラクターを入れる

> ⚠️ キャラクターイラストがないと「普通すぎる」デザインから脱せない。
> イラストを先に用意してからレイアウトを組む。
