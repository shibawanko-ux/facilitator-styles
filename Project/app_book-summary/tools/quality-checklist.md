# 要約品質チェックリスト

## 📋 チェック項目

以下の項目を順次チェックし、高品質な要約を完成させます。各項目について「完了」「要修正」「未着手」のいずれかで評価してください。

---

## 🔍 全体構成チェック

### 1. 基本情報
- [ ] **本の基本情報が正確に記載されている**
  - タイトル、著者、出版社、出版年
  - 要約作成日、作成者名
- [ ] **目次構成が適切に記載されている**
  - 章タイトルが正確
  - 章の順序が正しい

### 2. 全体要約
- [ ] **3つのクリティカル・メッセージが明確**
  - 本のエッセンスを的確に表現
  - 実務家にとって価値のある内容
- [ ] **想定読者が明確に定義されている**
  - デザイナー、ストラテジスト、組織開発者等
  - 読後に得られる視点・変化が具体的

---

## 📖 章別要約チェック

### 3. 理論の背景（WHY）
- [ ] **400〜500字で丁寧に説明されている**
  - 著者の問題意識が明確
  - テーマの重要性が説明されている
  - なぜそのテーマが重要なのかが分かる

### 4. キー概念と図式整理（WHAT）
- [ ] **表形式での概念整理が適切**
  - 重要な概念が漏れなく抽出されている
  - 説明が分かりやすく、実務的
  - 表の構成が論理的
- [ ] **矢印図での関係性整理が有効**
  - 概念間の関係性が明確
  - 視覚的に理解しやすい
  - 実務への応用がイメージできる

### 5. 実務での使い方と問い（HOW）
- [ ] **具体的な活用方法が提示されている**
  - ワークショップでの活用例
  - 対話設計への応用
  - プロジェクト運営での活用
- [ ] **効果的な問いかけ例が提供されている**
  - 実践的な問いかけ
  - 深い気づきを促す問い
  - 実務への応用を促す問い

### 6. 拓也への問いかけ
- [ ] **自分ごととして考えられる問いが設定されている**
  - 読者が「自分ごと」として考えられる
  - 実務への応用を促す内容
  - 深い気づきを促す問い

### 7. ハイライト一覧
- [ ] **ハイライト情報が適切に活用されている**
  - ピンク・青のハイライトが重視されている
  - 要約内容との対応関係が明確
  - 補強・裏付け情報として機能している

---

## 🎯 実務性チェック

### 8. 対象読者への適合性
- [ ] **デザイナー向けの内容**
  - UX・サービス・組織デザインへの応用
  - デザイン思考の視点
- [ ] **ストラテジスト向けの内容**
  - 戦略構築への応用
  - 組織設計への活用
- [ ] **組織開発者向けの内容**
  - 組織開発・設計への応用
  - ワークショップ・ファシリテーションへの活用

### 9. 実践可能性
- [ ] **具体的な実践方法が提示されている**
  - 明日から使える方法
  - 段階的な実践ステップ
  - 注意点・コツの提示

---

## 📝 文章品質チェック

### 10. 読みやすさ
- [ ] **初心者にも理解しやすい説明**
  - 専門用語の適切な説明
  - 分かりやすい例示
  - 段階的な説明
- [ ] **プロにも響く構造的な視点**
  - 深い洞察
  - 理論的な整理
  - 実務的な応用

### 11. 論理性
- [ ] **全体構成が論理的**
  - 章間の関連性が明確
  - 流れが自然
  - 結論が明確

### 12. 正確性
- [ ] **内容が正確**
  - 著者の主張を正確に反映
  - 誤解を招く表現がない
  - 適切な引用・参照

---

## 🔧 技術的チェック

### 13. フォーマット
- [ ] **Markdown形式が適切**
  - 見出しレベルの適切な使用
  - 表の形式が正しい
  - リストの形式が統一
- [ ] **視覚的な分かりやすさ**
  - 適切な空白・区切り
  - 読みやすい構成
  - 情報の階層が明確

### 14. メタデータ
- [ ] **作成情報が適切に記載**
  - 作成日・更新日
  - 作成者名
  - バージョン情報
- [ ] **参考資料が記載されている**
  - 使用したプロンプト
  - 参考資料
  - 品質チェック結果

## 📊 総合評価

### 評価基準
- **A（優秀）**: 全項目完了、高品質
- **B（良好）**: 主要項目完了、一部要改善
- **C（要改善）**: 複数項目で要修正
- **D（未完了）**: 多くの項目が未着手

### 最終チェック
- [ ] **総合評価**: [A/B/C/D]
- [ ] **優先改善項目**: [具体的な改善項目]
- [ ] **次回改善方針**: [改善の方針]

- [ ] 実務に有害な誤った情報
- [ ] 読者に誤解を招く表現
- [ ] 著作権侵害の可能性
