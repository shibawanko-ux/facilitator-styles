## [完了] ページ崩れ修正・ペルソナUX改善

### 問題の根本原因
- `/survey` と `/survey/thanks` が Layout なし → サイドバーが消える
- DashboardPage が admin 向け統計を全ロールに表示（RLS で 0件表示になり混乱）
- `SurveyAnswerPage` / `SurveyThanksPage` の `min-h-screen` が Layout 内で干渉

### 修正内容

| ファイル | 変更内容 |
|---|---|
| `App.tsx` | `/survey` と `/survey/thanks` を `<Layout>` で囲む |
| `SurveyAnswerPage.tsx` | `min-h-screen` を除去、早期returnの各状態も修正 |
| `SurveyThanksPage.tsx` | `min-h-screen bg-ad-bg` を除去 |
| `DashboardPage.tsx` | `useAuth` 追加、`role !== 'admin'` の場合は簡易ダッシュボード（アンケート回答・チームの結果カード）を表示 |

### ペルソナ別の体験
| ロール | `/` ダッシュボード | `/survey` |
|---|---|---|
| admin | チーム数・メンバー数・アンケート統計 | N/A（サイドバーにリンクなし） |
| leader | アンケート回答・結果カード（2列） | サイドバーあり ✅ |
| member | アンケート回答・結果カード（2列） | サイドバーあり ✅ |

- ビルド確認（5回目）：✅ 成功
