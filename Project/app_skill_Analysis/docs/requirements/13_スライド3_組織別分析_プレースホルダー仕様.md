# 13 スライド3 組織別分析 プレースホルダー仕様

スライド3（組織別分析）のレイアウト **analysis_by_organization** で使用するプレースホルダーとデータソース・表記ルールを規定する。  
従来の `{{O_block_1_body}}` は**廃止**し、代わりに組織人数・組織別スキル分析テーブル・組織別WS理解度を追加する。

## 1. LAYOUT ID

- **analysis_by_organization**（スライドマスタと一致。従来の `organization_analysis` から変更）

## 2. プレースホルダー一覧（組織1枚あたり）

| プレースホルダー | データ内容 | データソース・算出方法 |
|-----------------|------------|------------------------|
| `{{O_overall_score_name}}` | 組織名（例: デザイン室） | 02_組織別分析.csv（部署列） |
| `{{O_score}}` | 総合スコア（数値のみ。単位「点」なし） | 02_組織別分析.csv（総合スコア列） |
| `{{O_respondents_1}}` | 該当組織の人数（**数字のみ、単位なし**。例: 12） | 02_組織別分析.csv（人数列） |
| `{{OgrA_1}}`〜`{{OgrE_5}}` | 該当組織のスキル分析テーブル（下記 3. 参照） | pre_data / post_data から都度算出（02は拡張しない） |
| `{{OgrF_1}}`〜`{{OgrF_5}}` | 該当組織の**総合スコア**（実施前・直後・1ヶ月後・変化量(直後)・変化量(1ヶ月後)） | pre_data / post_data の該当組織 total から算出。17_スキル分析テーブル_総合スコア行追加要件 参照 |
| `{{OrkA_1}}` | 該当組織のWS理解度の平均（**単位「点」なし、小数点0.00まで**。例: 4.25） | post_data を所属部署でフィルタし、WS理解度の平均を算出 |
| `{{O_block_2_body}}` | 強みコメント | 02_組織別分析.csvから算出（10_分析コメント品質要件 参照） |
| `{{O_block_3_body}}` | 弱みコメント | 02_組織別分析.csvから算出（10_分析コメント品質要件 参照） |

### 廃止したプレースホルダー

- ~~`{{O_block_1_body}}`~~ … **削除**。本仕様の適用に伴い使用しない。関連ロジック（generate_department_characteristics 等）も削除する。
- **旧要件の要約**: `{{O_block_1_body}}` は、組織ごとに類似した定型文を避け、各組織の特徴に応じた差別化された分析コメントを生成するための要件だったが、廃止に伴い組織人数・組織別スキル分析テーブル・組織別WS理解度（`{{O_respondents_1}}`・`{{OgrA_1}}`〜`{{OgrE_5}}`・`{{OrkA_1}}` 等）に置き換えられている。

## 3. 組織別スキル分析テーブル（Ogr*）

- **表示内容**: スライド2（全社スキル分析テーブル）と同様に「実施前・直後・変化量」まで表示する。
- **命名規則**: スキル軸 A=具体化・検証力, B=リサーチ・分析力, C=構想・コンセプト力, D=伝達・構造化力, E=実現・ディレクション力, **F=総合スコア**（17 参照）。  
  - `_1` = 実施前, `_2` = 直後, `_3` = 1ヶ月後（Phase3）, `_4` = 変化量(直後), `_5` = 変化量(1ヶ月後)（Phase3）。
- **データ取得**: **02_組織別分析.csv は拡張しない**。  
  - 実施前: `analyze_by_department(pre_data)` の該当部署の各軸スコア  
  - 直後: `analyze_by_department(post_data)` の該当部署の各軸スコア（または 02 の列から取得）  
  - 変化量(直後): 直後 − 実施前。Phase3 の場合は 1ヶ月後・変化量(1ヶ月後) も同様に算出。

## 4. 表記ルール

- **OrkA_1**（組織別WS理解度の平均）: 単位「点」は付けない。小数点は 0.00 まで（例: `4.25`）。
- **O_respondents_1**（組織の人数）: 数字のみ、単位なし（例: `12`）。
- **O_score**: 従来どおり数値のみ、単位「点」は付けない（05_分析と出力仕様 2.5 参照）。

## 5. Phase 2 / Phase 3

- Phase 2: 上記のうち実施前・直後・変化量(直後) を出力。1ヶ月後・変化量(1ヶ月後) は `-` または未使用。
- Phase 3: 1ヶ月後・変化量(1ヶ月後) もデータがある場合に算出・出力する。
