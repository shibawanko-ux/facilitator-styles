# 01 目的と評価の位置づけ

## 1. 背景と課題

- 従来の研修報告は「受講直後の満足度・感想」に留まり、**研修投資の対効果（ROI）が可視化されていない**という課題がある。
- 本アプリは、研修による**行動変容**と**スキル定着**を定量的に測定し、データドリブンな人材育成を実現することを目的とする。

## 2. 評価のレベル（カークパトリックモデル）

| レベル | 名称 | 本施策での扱い |
|--------|------|----------------|
| Level 1 | 反応 | 受講満足度（実施済。直後アンケートの満足度・理解度・NPS等で測定） |
| Level 2 | 学習 | スキル習得度・理解度（本施策で測定。PreWS→PostWSの変化） |
| Level 3 | 行動 | 現場での実践・定着度（本施策で測定。Follow-up・上長評価で検証） |

## 3. 評価の特徴

- **職能要件に基づいた評価軸**: 組織の期待値と整合した成長指標を可視化する。
- **行動基準による評価**: 自己評価のハードルを下げつつ、実態を捉える。
- **多角的な測定**: 本人評価と上長評価を組み合わせ、定着度の客観性を担保する。

## 4. 本プロジェクトの価値

1. **ROIの可視化**: 研修投資がスキル定着にどの程度結びついたかを定量的に測定する。
2. **組織の特性把握**: 強み・弱みを明確にし、戦略的な人材育成を可能にする。
3. **個別フォロー**: 「知識止まり」の層を特定し、効果的なサポートを提供する。
4. **継続的改善**: データに基づいた育成ロードマップの策定に活用する。
