# 02 評価フレームワーク

## 1. 5つの評価軸

ビジネスの課題解決プロセスを、以下の5つのスキル軸に分解して測定する。各軸は**3問ずつ、計15問**（Q1〜Q15）で測定する。

| 順序 | 評価軸（日本語） | 英語キー | 測定する能力 |
|------|-----------------|----------|--------------|
| 1 | リサーチ・分析力 | research | 事実に基づき、本質的な課題を発見する力 |
| 2 | 構想・コンセプト力 | concept | 独自の切り口で、解決策の方向性を定義する力 |
| 3 | 具体化・検証力 | delivery | アイディアを形（プロトタイプ）にし、検証する力 |
| 4 | 伝達・構造化力 | communication | 複雑な情報を整理し、論理的に合意を得る力 |
| 5 | 実現・ディレクション力 | implementation | 実現可能性を見極め、周囲を巻き込んで完遂する力 |

**質問と軸の対応**:
- Q1〜Q3: research
- Q4〜Q6: concept
- Q7〜Q9: delivery
- Q10〜Q12: communication
- Q13〜Q15: implementation

## 2. 5段階レベル定義

| 点数 | レベル定義 | 説明 |
|------|-----------|------|
| 1点 | できない / 知らない | 基礎知識・経験がない状態 |
| 2点 | 助けがあればできる（要サポート） | 指導があれば実行可能 |
| 3点 | 一人でできる（標準・自走レベル） | 自立して実行できる |
| 4点 | 工夫・応用してできる | 状況に応じて応用可能 |
| 5点 | 人に教えられる（指導レベル） | 他者に指導できるレベル |

スコアは**1〜5の整数（または小数点含む）**で、この範囲外の値は無効とする。

## 3. スコア算出方法

### 3.1 個人の軸スコア

- **各評価軸のスコア** = その軸に属する3問の**平均**（欠損は除いて計算）。
- 例: リサーチ・分析力 = (Q1 + Q2 + Q3) / 3（有効な回答数の平均）。

### 3.2 個人の総合スコア

- **総合スコア** = 全15問の合計点 ÷ 15（有効回答に基づく平均）。

### 3.3 組織スコア

- **組織の軸スコア** = 全員のその軸スコアの合計 ÷ 人数。
- **組織の総合スコア** = 全員の総合スコアの合計 ÷ 人数。

### 3.4 上長評価（複数評価者）

- 対象者ごとに、各軸（M1〜M5）について**中央値**で集計する。
- 重み付きを利用する場合は、重み付き中央値で集計する。

## 4. 解釈の目安

- **1.0〜2.0**: 基礎的な理解・経験が必要な段階
- **2.1〜3.0**: サポートがあれば実行可能な段階
- **3.1〜4.0**: 標準レベルで実行できる段階（目標レベル）
- **4.1〜5.0**: 応用・指導が可能な高度な段階
