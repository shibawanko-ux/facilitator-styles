# 03 実施プロセス

3つのフェーズで定点観測を行い、**成長の軌跡**と**定着の真偽**を判定する。

```
PreWS（実施前） → PostWS（直後） → Follow-up（1ヶ月後）
```

## 1. PreWS（実施前）

| 項目 | 内容 |
|------|------|
| 対象者 | 本人のみ |
| 目的 | 現状のスキルレベル（Baseline）の把握 |
| 内容 | 評価フレームワークに基づく自己評価アンケート（15問）、各軸1〜5点 |
| タイミング | ワークショップ実施前 |

## 2. PostWS（直後）

| 項目 | 内容 |
|------|------|
| 対象者 | 本人のみ |
| 目的 | 学習内容の理解度、活用意欲の確認 |
| 内容 | 同じ15問の再評価＋満足度・理解度・NPS・活用意欲等（オプション） |
| タイミング | ワークショップ終了直後または翌日 |

**注意**: PostWSのスコア上昇は「理解した」ことを示すが、「実践できる」ことを保証するものではない。Follow-upで定着を確認する。

## 3. Follow-up（1ヶ月後）

| 項目 | 内容 |
|------|------|
| 対象者 | 本人＋上長 |
| 目的 | 現場での実践頻度、行動変容、上長からの客観評価 |
| 本人 | 同じ15問の再評価＋実践頻度（Q16B等） |
| 上長 | 対象者ごとに各軸（M1〜M5）を1〜5点で評価 |
| タイミング | ワークショップ実施から1ヶ月後 |

## 4. 測定のポイント

- **成長の軌跡**: PreWS → PostWS → Follow-up の3点でスキル推移を可視化する。
- **定着の真偽**: 1ヶ月後の上長評価を加え、自己評価と実態のギャップを検証する。
- **行動変容**: 知識として理解しただけでなく、現場で活用できているかを測定する。
