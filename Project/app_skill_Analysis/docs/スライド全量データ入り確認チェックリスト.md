## 実際のスライドで確認してほしいこと

1. **残っている `{{...}}` がないか**  
   スライドを1枚ずつ開き、`{{` で検索して、プレースホルダーがそのまま表示されていないか確認する。
2. **レーダーチャート**  
   スライド2のレーダーチャート用の位置に、  
   `reports/生成レポート_アイフル株式会社_リサーチWS_Phase2_レーダーチャート.png` が挿入されているか確認する。
3. **組織別スライド**  
   7組織分のスライドが存在し、各スライドの部署名・スコア・表・強み・伸びしろが、スライド挿入内容の該当組織と一致しているか確認する。

以上で、**データの全量**と**スライド上の表示**の両面から確認できます。
