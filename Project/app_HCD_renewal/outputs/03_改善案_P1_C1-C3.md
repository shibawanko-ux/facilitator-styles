# 改善案：プロジェクト1 C群（導入推進：C3）

## ■ C3｜人材育成能力

### 【勉強会の設計と実施】
・プロジェクトで得たプロセスと学びを、他部署のメンバーにも展開するため、勉強会を企画・実施した。
・勉強会では、HCDプロセスの概念を抽象的に説明するのではなく、今回のプロジェクトの進め方をHCDプロセスと対比させる形で構成した。
・「実際のプロジェクトではどのフェーズで何をしたか」をHCDサイクル上に位置づけて示すことで、参加者が自分たちの業務との接点をイメージしやすい内容とした。

### 【資料の作成】
・勉強会の内容をもとに、HCDプロセスと今回のプロジェクトの進め方を対比的に整理した資料を作成した。
・抽象的なフレームワークとして伝えるのではなく、実践事例と並べて示すことで、他部署が再現可能な形に整理した。
