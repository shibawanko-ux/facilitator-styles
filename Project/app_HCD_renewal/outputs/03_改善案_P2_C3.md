# 改善案：プロジェクト2 C群（導入推進：C3）

## ■ C3｜人材育成能力

### 【振り返りを通じたHCDプロセスの定着】
・プロジェクト終了後、参加者とともに丁寧な振り返りを実施した。
・振り返りでは「何が起きていたか」だけでなく、「プロセスとしてどこに戻ったか」「なぜ戻ったのか」を言語化しながら、その時々の判断を思い起こす形で進めた。
・HCDのサイクル図をもとにプロセスを振り返ることで、「なぜあの場面で戻ることが必要だったのか」というプロセスの意味づけを参加者自身が言語化できる状態をつくった。
・振り返りを通じて、HCDプロセスは難しいものではなく、「判断しながら行き来するもの」という認識を参加者が自ら獲得できる場を設計した。
・人事部が本プロジェクトに込めていた「デザインの考え方を組織として取り入れたい」という意向に対し、振り返りを通じて参加者が自分の言葉でプロセスを語れるようになったことで、デザインの考え方が組織に根付く入口をつくることができた。
