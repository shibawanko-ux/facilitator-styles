# サンプル・参照ファイルの配置

本フォルダには、要件定義で参照する次のファイルを配置する。

## 配置するファイル

1. **最終サポートのサンプル**  
   - ファイル名例: `20260130_アイフル株式会社御中_ワークショップ振り返り.key.pdf`  
   - 用途: 文言をアプリで作成するスライド（P3, P9〜P15）の構成・レイアウトの参照。クライアント提出用振り返り資料の完成形サンプル。

2. **ワークショップメニュー**  
   - ファイル名例: `Workshopメニュー.key.pdf`  
   - 用途: ⑨ 各種支援プログラム文で推薦するプログラムの一覧・分類（イチオシ・おすすめ・候補）およびプログラム詳細の出典。

## 配置方法

上記2つの PDF をこのフォルダ（`docs/samples/`）にコピーする。  
要件定義書 [01_requirements.md](../01_requirements.md) の 3.3・3.4 では、配置後のパスを `docs/samples/xxx.pdf` として参照する。
