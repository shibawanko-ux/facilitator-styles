# app_Client_workshop_report — プロジェクトメモ

## 概要
クライアント向けワークショップレポートを生成するアプリ。

## 構造
```
Project/app_Client_workshop_report/
├── scripts/        ← 生成スクリプト（Python）
├── projects/       ← プロジェクトデータ
├── docs/           ← ドキュメント
└── requirements.txt← Python依存パッケージ
```

## 起動・実行
```bash
cd Project/app_Client_workshop_report
pip install -r requirements.txt
python scripts/<スクリプト名>.py
```
