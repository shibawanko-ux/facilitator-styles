# 各ファシリテーターの所感

## 外舘
ワークを通してかなりの熱量で取り組んでくださっていました！それだけチーム内でのコミュニケーションが普段から取れているのだと感じました。その熱量をコントロールし、さらなる成果に繋げるために、ぜひゴールと目的を掲げた場の設計を見直し、ファシリをチームの文化にしていってください。

## たくや
場を救うのは隠れファシリテーターです。今回同じ体験をしたからからこそ隠れファシリテーターとなれる人たちなので、会議などにあえて呼んで場を動かすようにすすめてみてはどうでしょうか？ファシリテーターの楽しさがそこでわかると思いますし、その楽しさを伝播していってください

## くろ
ファシリテーターは中立な立場で場に働きかけることが重要です。参加者がみんな声を出せているか、参加者が作る場の雰囲気がどのようになっているかなど、メタ的視点で場を見れるようになるとファシリテーションの理解をより深めながら、楽しんでいけると思います！少しずつ実践していってください！

## shimaken
ファシリテーターは中立であることだけが大事というより、議論を円滑にかつ深めながら、より良いゴールに向かってはしらせていくことが肝心です。自身が常に中立に立つべきというより、議論の状態、関係者同士の状況、場の雰囲気を俯瞰的に観察して、どう場に関わるかが大切です。難しいですが、「入り込む」のと「ひく」動きを大切に。
