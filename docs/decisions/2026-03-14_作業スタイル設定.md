# 決定事項：作業スタイル設定

## 日付
2026-03-14

## 決定内容

### 自動ファイル作成
- 実装・機能追加の開始時 → 要件定義ファイルを自動生成（`docs/requirements/`）
- 会話の中で決定事項が出たとき → 決定事項メモを自動保存（`docs/decisions/`）
- やることが複数出てきたとき → タスクリストを自動整理（`docs/tasks/`）

### 会話スタイル
- 作業の節目ごとに確認を入れる
- 選択肢は A / B 形式で提示する
- 完了後に次のアクションを提案する
- 判断理由を都度共有する

## 背景
ユーザーがCursorのように自動でドキュメントを生成し、一緒に丁寧に作業を進めるスタイルを希望。
