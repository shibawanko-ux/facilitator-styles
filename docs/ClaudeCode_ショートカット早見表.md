# Claude Code ショートカット・コマンド早見表

---

## スラッシュコマンド（`/` で一覧表示）

| コマンド | 説明 |
|---------|------|
| `/clear` | 新しいセッションを開始 |
| `/resume` | 過去のセッションを再開 |
| `/memory` | CLAUDE.md・メモリを表示・編集 |
| `/config` | 設定メニューを開く |
| `/cost` | トークン使用料金を確認 |
| `/vim` | Vim モードを有効化 |
| `/loop 5m xxx` | 定期タスクの繰り返し実行（例：`/loop 5m デプロイ確認`） |
| `/voice` | ボイスモード（push-to-talk） |
| `/effort` | 処理レベルを low / medium / high で設定 |
| `/hooks` | フック設定メニュー |
| `/skills` | スキル管理メニュー |

---

## キーボードショートカット

| キー | 説明 |
|-----|------|
| `Ctrl+C` | キャンセル |
| `Ctrl+D` | セッション終了 |
| `Ctrl+L` | 画面クリア（会話履歴は保持） |
| `Ctrl+T` | タスクリスト 表示/非表示 |
| `Ctrl+O` | ツール使用の詳細を 表示/非表示 |
| `Ctrl+G` | 外部エディタでプロンプトを編集 |
| `Ctrl+B` | 実行中の Bash コマンドをバックグラウンド化 |
| `Shift+Tab` | パーミッションモード切り替え |
| `Esc` + `Esc` | 直前の操作を巻き戻す |

---

## ファイル指定・参照

| 操作 | 説明 |
|-----|------|
| `@` + ファイル名 | ファイルパスのオートコンプリート（作業ディレクトリ配下が対象） |
| `@app.ts#5-10` | 特定の行範囲を指定して参照 |
| `Option+K`（Mac）/ `Alt+K`（Win） | 選択中のコードのファイルパス＋行番号を自動挿入（VS Code） |
| `Shift` + ドラッグ&ドロップ | ファイルをチャット欄に添付（VS Code） |

## CLI コマンド（ターミナルから）

| コマンド | 説明 |
|---------|------|
| `claude` | セッション開始 |
| `claude -c` | 最新セッションを再開 |
| `claude --version` | バージョン確認 |
| `claude update` | 最新版にアップデート |
| `claude auth login` | ログイン |
